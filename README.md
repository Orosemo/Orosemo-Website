## general infos

This is a rework of my old  personal website (which was made with HTML) because i wanted to have translations and themes.
I decidet to make it with react and Tanstack router because i wanted to try out react for a while now and i thought it was a good opportunity.

You are free to use any parts of this website for your personal projects but please dont just copy the whole thing ;D

I used Ai for the translations and research

### what i learned

I learned that commiting every change is very worth it if you change your mind about some features and dont want to revert the changes by hand.
Also i learned a lot about react and TS with this project.

## features

### themes

My website supports multiple themes (light, dark and two seasonal ones).

### translations

My website now contains translations for the 50 most spoken languages which is great for accecebility.

### project search functions

My websites games and tools page now have search bars (something which wasnt so easy to implement with HTML).
